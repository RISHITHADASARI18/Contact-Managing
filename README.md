# 📞 Contact Manager

A simple and interactive Contact Management System built using Python and Tkinter.  
This application allows users to store, manage, and delete contacts with a clean GUI.

---

## 🚀 Features

- 🌍 Country-based phone number validation  
- 📱 Real-time digit limit based on selected country  
- ➕ Add new contacts  
- ❌ Delete selected contacts  
- 📋 View contacts in a scrollable list  
- 🎨 Clean dark-themed user interface  

---

## 🛠️ Tech Stack

- Python  
- Tkinter (GUI)
